Para buildear y correr la app:

- Version browser (para testear rapido):

ionic cordova build browser

ionic cordova run browser

- Version android

ionic cordova build android

ionic cordova run android (necesario tener un emulador instalado)

ionic cordova emulate android (si el emulador ya esta corriendo)

Para probar en el telefono directamente (luego de buildear), la apk se encuentra en el path: hubin-app\platforms\android\app\build\outputs\apk\debug
